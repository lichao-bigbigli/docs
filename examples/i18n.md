# 国际化示例

为你的 VitePress 站点添加多语言支持。

## 第 1 步：配置语言

更新 `.vitepress/config.ts`：

```typescript
import { defineConfig } from 'vitepress'

export default defineConfig({
  locales: {
    root: {
      lang: 'zh-CN',
      title: 'VitePress',
      description: '中文文档'
    },
    en: {
      lang: 'en-US',
      title: 'VitePress',
      description: 'English documentation'
    }
  },
  themeConfig: {
    nav: [
      { text: '简体中文', link: '/', activeMatch: '^/(?!en/)' },
      { text: 'English', link: '/en/', activeMatch: '^/en/' }
    ],
    sidebar: {
      '/': [/* 中文侧边栏 */],
      '/en/': [/* 英文侧边栏 */]
    }
  }
})
```

## 第 2 步：创建语言目录

```bash
mkdir -p en/guide
mkdir -p en/reference
```

## 第 3 步：创建翻译内容

创建 `en/index.md`：

```md
---
home: true
hero:
  name: VitePress
  text: Vite & Vue Powered Static Site Generator
  tagline: Get started in minutes
  actions:
    - theme: brand
      text: Get Started
      link: /en/guide/getting-started
---
```

创建 `en/guide/getting-started.md`：

```md
# Getting Started

Welcome to VitePress! This guide will help you get started quickly.
```

## 第 4 步：添加语言切换

在导航栏中添加语言切换：

```typescript
themeConfig: {
  nav: [
    { text: '简体中文', link: '/', activeMatch: '^/(?!en/)' },
    { text: 'English', link: '/en/', activeMatch: '^/en/' }
    // 其他导航项...
  ]
}
```

## 第 5 步：测试你的站点

启动开发服务器：

```bash
npm run docs:dev
```

你应该能在中文与英文之间切换。

## 进阶配置

### 默认语言

设置默认语言：

```typescript
export default defineConfig({
  lang: 'zh-CN',
  locales: {
    root: { lang: 'zh-CN' },
    en: { lang: 'en-US' }
  }
})
```

### RTL 支持

对从右到左的语言（如阿拉伯语）：

```css
[dir="rtl"] {
  direction: rtl;
  text-align: right;
}
```

## 核心概念

1. **locales**：定义语言配置
2. **目录结构**：组织翻译内容
3. **语言切换**：允许用户切换语言
4. **RTL 支持**：支持从右到左语言
