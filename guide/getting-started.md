# Getting Started

Welcome to BigBigLi! This guide will help you get started quickly.

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js (v18 or higher)
- npm or yarn

## Installation

Follow the [Installation](/guide/installation) guide to install BigBigLi.

## Quick Start

After installation, you can start using BigBigLi with the following command:

```bash
bigbigli start
```
