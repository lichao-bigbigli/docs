# Installation

Learn how to install BigBigLi on your system.

## Using npm

```bash
npm install bigbigli
```

## Using yarn

```bash
yarn add bigbigli
```

## Using pnpm

```bash
pnpm add bigbigli
```

## Verify Installation

After installation, verify it by running:

```bash
bigbigli --version
```

## Troubleshooting

If you encounter any issues during installation, check the following:

- Make sure Node.js is installed and up to date
- Check your network connection
- Try clearing your npm cache: `npm cache clean --force`
